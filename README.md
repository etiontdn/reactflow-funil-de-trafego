# FunnelFlow – Visual Funnel Strategy Tool

**FunnelFlow** é uma ferramenta interativa para modelagem e simulação visual de estratégias de funis de marketing e vendas. Desenvolvida com **Next.js 14**, **XYFlow (React Flow)** e **Tailwind CSS**, a aplicação permite que estrategistas desenhem fluxos complexos com cálculos de conversão em tempo real.

## 🛠️ Tecnologias Utilizadas

* **Framework:** [Next.js 14 (App Router)](https://nextjs.org/)
* **Engine de Diagramação:** [XYFlow (React Flow)](https://reactflow.dev/)
* **Estilização:** [Tailwind CSS](https://tailwindcss.com/) & [Shadcn/ui](https://ui.shadcn.com/)
* **Ícones:** [Hugeicons](https://hugeicons.com/)
* **Linguagem:** TypeScript

## 📦 Instalação e Execução

Para rodar o projeto localmente, siga os passos abaixo:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/seu-usuario/funnel-flow.git
    ```

2.  **Instale as dependências:**
    ```bash
    npm install
    ```

3.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```

4.  **Acesse no navegador:**
    `http://localhost:3000`

## 🏗️ Estrutura de Arquivos

* `/app`: Rotas e layout principal da aplicação.
* `/components/nodes`: Implementação dos nós customizados e lógica de processamento de dados.
* `/components/ui`: Componentes de interface baseados em Radix UI.
* `/lib`: Funções utilitárias e configurações globais.
